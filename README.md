# Swarm Thinking — 群体智能头脑风暴法

> 从 [MiroFish](https://github.com/666ghj/MiroFish) 多智能体社会模拟引擎提炼的思维方法论

## 核心思想

当面对复杂决策/创意问题时，人的思维容易陷入单一视角的盲区。此方法通过**构建多个人格化的利益相关方，让他们在模拟推演中相互碰撞、辩论、博弈，最终从涌现的群体行为中找到最优解。**

## 适用场景

- 🧠 头脑风暴 — 创意发散、产品方向评估
- 📊 决策分析 — 方案对比、风险评估
- 🔮 未来推演 — 策略沙盘、情景模拟

## 方法流程（6 步）

1. **需求澄清** — 确认推演类型（决策/创意/预测）
2. **人格构建** — 为每个利益方创建独立人格卡
3. **4 维发散** — 时间线、成本收益、连锁反应、极端场景
4. **收敛分析** — 提取共识区、分歧区、盲区
5. **方案归因** — 矩阵排序 + 最优方案推荐
6. **策略简报** — 最终交付物
